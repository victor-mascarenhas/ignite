# Ignite

It's a front-end project to fetch data from [rawg.io](https://rawg.io/apidocs) API and display the upcoming, new and popular games in the database.

Production Deployment:
https://ignite-phi.vercel.app/

## Used dependencies:

    axios
    framer-motion
    react
    react-redux
    react-router-dom
    styled-components
